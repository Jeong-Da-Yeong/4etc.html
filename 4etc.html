*깃허브 이용방법*
  1. new repositary(새로운 저장소) 선택
  2. repositary name(저장소 이름)을 꼭 자신의 html이 들어있는 폴더명으로 하기
  3. public으로 체크, initialize this repository ~ 체크 후 create repositary (저장소 생성하기) 클릭
  4. upload files클릭 후 드래그해서 html파일 전부 업로드 후, commit changes에 first version등과 같 보관된 내용적기
  5. setting 클릭 후, Git hub pages에 sourse를 none에서 master branch로 변경한 후 save
  6. Github pages에서 http:/주소를 클릭하면 주소가 나온다
  7. 주의할점!: 가장먼저 표시될 html을 index.html로 수정해준다
